Styled Components

Apollo Client

GraphQL Yoga

GraphQL

Prisma

Jest

Enzyme

## Installing Dependencies

From within the root directory:
```
npm install
```
To run server Front end
```
npm run dev
```
To run server Back end
```
npm run start
```
To run tests
```
npm test
```
